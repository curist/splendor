# Splendor

[Splendor on Board Game Geek](https://boardgamegeek.com/boardgame/148228/splendor)

## Development

```sh
npm i
npm start
# open http://localhost:8080
```

## Test

```sh
npm t
```

## Build

```sh
npm i
npm run build
# bundled app sits in ./build/
```

# AI interfaces

check `app/AI/ai_dumb.js` for a minimal working implementation.
